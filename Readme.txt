# Cyber Cafe Management System:

Developed a web-based system to manage user activity, login details, and session reports for cyber caf√©s.
Used PHP, MySQL, and JavaScript for backend and frontend functionality.


Steps to run the Project:

1. Clone the repository
2. Install XAMPP
3. Move the project folder to the htdocs directory
4. Start Apache and MySQL from the XAMPP control panel
5. Import the database file (.sql) into phpMyAdmin
6. Open the project in your browser at http://localhost/your-folder-name
